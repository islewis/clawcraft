# Mineflayer Minecraft Bot Project

## Project Overview
This project creates a Minecraft bot using the [Mineflayer](https://github.com/PrismarineJS/mineflayer) library.

- **Target Minecraft Version**: 1.21.11
- **Language**: JavaScript (Node.js)
- **Main Library**: mineflayer

## Development Guidelines

### Code Style
- Use JavaScript (no TypeScript unless explicitly requested)
- Keep code simple and focused — avoid over-engineering
- Follow existing patterns in the codebase

### Dependencies
- Primary: mineflayer for Minecraft bot functionality
- Use npm for package management

### Task Approach
- Only make changes that are directly requested
- Don't add features beyond what's asked
- Don't add extensive comments or refactoring unless explicitly requested
- Verify Mineflayer API compatibility with 1.12.4 when implementing features
- Keep helper output compact (newlines ok, minimize blank lines/filler) — output will be ingested by AI

### Git Workflow
- Create meaningful commits with clear messages
- Don't force-push unless explicitly authorized
- Ask before pushing to remote repositories
